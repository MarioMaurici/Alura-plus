# Alura PLUS

Projeto feito através do curso front-end da alura. O Alura PLUS é uma página feita mostrando os planos para assinar a plataforma da alura



## ✔️ Técnicas e tecnologias utilizadas

- `HTML`: 
    - `section`
    - `class`
- `CSS`:
    - `:hover`;
    - `flexbox`;
    - `grid`;
